MTA GTFS npm package

View on Github →https://www.npmjs.com/package/mta-gtfs →

I needed real-time subway data for a project I was working on so I created a Node.js library to access the real-time MTA subway train schedule API.

It powers a subway widget in the current version of Mirror and has been downloaded ~2k times via npm.
